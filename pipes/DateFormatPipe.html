Description

Core pipe to format date based on date format set by the user.

Example:

<!-- Renders 7/12/2023 for shortDateFormat users -->
<span>{{2023-07-12T11:58:32.325 | dateFormat | async}}</span>

Metadata

Name dateFormat

Methods

transform
transform(value: Date | number | string, format: string, timezone?: string, locale?: string)

The pipe picks date format in following manner:

  • If a format is provided, the pipe first checks if it matches date format present in user locale. If a match is found, the date format passed will be applied.
  • If no match is found in the user locale, the pipe will attempt to match the provided format with an entry in the DateFormat enum.
  • If no match is found in the DateFormat enum, then the pipe uses the exact date format passed.
  • If no date format is passed, then the pipe defaults to 'DateFormat.ShortDateFormat' which is 'M/d/yyyy' See DateFormat for more details on supported date formats. The pipe determines the locale in the following order:
  • If a locale is provided, it is used directly.
  • If not provided, the pipe will attempt to use the user's locale.
  • If the user's locale is not available, it defaults to 'en-US'.
