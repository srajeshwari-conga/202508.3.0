Description

The Document Selector component displays a list of documents with selection functionality. It provides a two-panel view: available documents on the left and selected documents on the right.

Preview

Usage


import { DocumentSelectorModule } from '@congarevenuecloud/elements';

@NgModule({
  imports: [DocumentSelectorModule, ...]
})
export class AppModule {}

```html
<apt-document-selector
  [attachmentList]="attachments"
  [selectedDocuments]="selectedDocs"
  (selectedDocumentsChange)="onDocumentsSelected($event)">
</apt-document-selector>
