Description

The date range component includes two date input fields: one for start date and one for end date. The input fields use the ngx-bootstrap date picker component.

By default the date range component will calculate the beginning and ending date for the cart item that is passed to it based on the frequency and term of the product. When the user selects a date in the start date or end date inputs the onStartDateChange and onEndDateChange methods will be fired respectively.

Usage

import { InputDateModule } from '@congarevenuecloud/elements';

@NgModule({
imports: [InputDateModule, ...]
})
export class AppModule {}
// Basic Usage.
```typescript
<apt-date-range-input
            [cartItem]="item"
            (onStartDateChange)="handleStartChange($event, item)"
            (onEndDateChange)="handleEndDateChange($event, item)"
></apt-date-range-input>
// Implementation with custom options.
```typescript
<apt-date-range-input
            (onStartDateChange)="handleStartChange($event, item)"
            (onEndDateChange)="handleEndDateChange($event, item)"
            [cartItem]="item"
            [disableStartDate]="disableStartDate"
            [disableEndDate]="disableEndDate"
            [readonly]="false"
></apt-date-range-input>

Inputs

bsConfig
Type : Partial<BsDatepickerConfig>

Configuration object for the ngx-bootstrap date picker that allows customization of various options. Default values:

  • showWeekNumbers: false
  • dateInputFormat: 'MM/DD/YYYY'
  • containerClass: selects calendarTheme
  • adaptivePosition: selects enableAutoPositioning

Any values provided in bsConfig will override the default settings. Partial<BsDatepickerConfig> makes all properties of BsDatepickerConfig optional, allowing a subset of properties to be provided.
calendarTheme
Type : "theme-green" | "theme-blue" | "theme-red" | "theme-orange" | "theme-dark-blue" | "theme-default"
Default value : 'theme-green'

The calendarTheme property allows you to customize the calendar's appearance by selecting one of the defined themes. Available themes:

  • 'theme-green': Green themed calendar.
  • 'theme-blue': Blue themed calendar.
  • 'theme-red': Red themed calendar.
  • 'theme-orange': Orange themed calendar.
  • 'theme-dark-blue': Dark blue themed calendar.
  • 'theme-default': Grey themed calendar.
cartItem
Type : CartItem

The cart item associated with the date range picker.
disableDays
Type : Array<number>
Default value : null

An array of numbers representing the days of the week to disable in the datepicker. The days of the week are indexed as follows:

  • Sunday = 0
  • Monday = 1
  • Tuesday = 2
  • Wednesday = 3
  • Thursday = 4
  • Friday = 5
  • Saturday = 6
disableEndDate
Type : boolean
Default value : false

Flag to disable the end date input.
disableStartDate
Type : boolean
Default value : false

Flag to disable the start date input.
displayDate
Type : "StartDate" | "EndDate" | "Both"

Flag to display a particular date.
displayDateFormat
Type : string
Default value : 'StandardDate'

A string representing the date format to be used for display. If no format is provided, the default date format will be 'StandardDate' format, which is 'MM/dd/YYYY'. This is used for fields where the date cannot be edited. See DateFormat for additional details on the date formats.
enableAutoPositioning
Type : boolean
Default value : true

Flag to enable or disable the auto positioning of the date picker. When enabled (true), the date picker will automatically adjust its position on the screen
hideLabel
Type : boolean
Default value : false

Flag to show/hide the label for the date picker.
label
Type : string

Overrides the default values for the input label.
maximumDate
Type : Date

The maximum date allowed to select in the date picker. Dates beyond the maximum date are disabled.
minimumDate
Type : Date

The minimum date allowed in the date picker. If not provided, it defaults to the start date.
readonly
Type : boolean
Default value : false

Flag to check if this component should be read only.
small
Type : boolean
Default value : false

Flag to apply small class on input field.

Outputs

Event emitter that fires when the end date input is changed.
onStartDateChange
Type : EventEmitter<any>

Event emitter that fires when start date input is changed.
