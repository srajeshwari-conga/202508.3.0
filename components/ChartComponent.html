Description

Chart component displays the data as a chart based on the configuration passed in to this component.

Inputs

amountsByStatus
Type : Object

Object used to display the total dollar amount within the tooltip for bar charts.
animationStyle
Type : AnimationStyle
Default value : { duration: 1000, easing: AnimationOptions.EASE_OUT_QUART, delay: 200, animateScale: false }

Allows users to outline the nature of the animation as defined in AnimationStyle interface.

  • duration default value is 1000 milliseconds
  • easing default value is easeOutQuart
  • delay default value is 200 milliseconds
  • animateScale default value is false

See AnimationStyle for more details on the structure and fields required.
centerText
Type : any

Value to be displayed in the center of chart.
chartTitle
Type : string

Title to be displayed for the chart.
colorPalette
Type : Array<string>

List of colors to be used on the chart.
currencyConversionCode
Type : string

A string value representing ISO 4217 currency code to convert the currency values to.
data
Type : Object

Dataset to be displayed on the chart.
labelDataTransform
Type : string

String to specify what the data on chart label needs to be transformed to. Can take values like 'currency' etc.
legendPosition
Type : "top" | "bottom" | "left" | "right"
Default value : 'top'

Position of legend on the chart.
showAnimation
Type : boolean
Default value : false

Allows users to add animations to charts, with different animation options defined in AnimationStyle interface. Defaulted to false See AnimationStyle for more details on the structure and fields required.
showLegend
Type : boolean
Default value : true

Flag to show/hide legend on the chart.
type
Type : string

Type of the chart to be rendered.
