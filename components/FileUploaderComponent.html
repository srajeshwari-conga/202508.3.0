Description

The FileUploaderComponent is a component designed for uploading files with configurable options for file types, size limits, quantity limits, and progress tracking. It emits file details after a successful upload and provides error messages when restrictions are not met.

Inputs

fileUploadLabel
Type : string

Label to be shown for file uploader template.
maxFileSize
Type : number
Default value : 29360128

Maximum file size allowed for upload, in bytes. The default value is set to 29360128 bytes (approximately 28 MB), This value is provided by the platform API. If the uploaded file exceeds this size, an error message will be displayed.

Preview
maximumFilesAllowed
Type : number
Default value : 25

Maximum number of files allowed for upload. This property limits the number of files a user can select and upload at once. Defaults to 25 if no value is provided.

Preview
showFileUploadProgress
Type : boolean
Default value : true

Flag defines if progress bar should be shown while file upload is in progress. Defaults to true.
supportedFileTypes
Type : string

A string containing comma-separated file extensions to be supported for file upload. For example: 'jpeg,png,pdf'. If a file with a different type is uploaded, the component will display an appropriate error message.

Preview

      </p>
    </div>

    <!-- message for supported file types -->
    <div class="mt-2" *ngIf="uploadedFiles == null && isSupportedFileType && supportedFileTypes?.length">
