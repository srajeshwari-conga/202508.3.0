Description

This component is a work in progress.

Constraint rule sidebar is used to show a list of all the constraint rule messages that have been applied to the current cart.

replaceProducts
replaceProducts(rules: Array<ConstraintRuleDetail>, actionItems: Array<ActionItem>, loadingKey: string)

Executes replacement action based on rule criteria. Rule action product will be replaced with condition product.

See ConstraintRuleDetail for more details on the structure and fields required. See ActionItem for more details on the structure and fields required.

Parameters :
Name Type Optional Description
rules Array<ConstraintRuleDetail> No
  • A list of constraint rule details that specify the replacement rules.
actionItems Array<ActionItem> No
  • A list of ActionItem interface representing the action items to be replaced.
loadingKey string No
  • A unique identifier to track the loading state of a replacement action. It sets to true when the process starts and false when it completes.
Returns : void
replaceSelectedProducts
replaceSelectedProducts(ruleIndex: number, loadingKey: string)

Replaces selected products based on the chosen rule, identified using 'ruleIndex', and the selected items. If there are selected products within the specified rule, they will be replaced accordingly.

Parameters :
Name Type Optional Description
ruleIndex number No
  • Index of the rule to be replaced.
loadingKey string No
  • A unique key to track the loading state of the replacement action. It is set to true when the process begins and false when it ends.
Returns : void
resetRulesConfiguration
resetRulesConfiguration()

Resets the rules configuration by initializing or clearing the errors array, disabling the replace button for all error entries, and resetting the checkbox status for each rule's action items if applicable.

Returns : void
