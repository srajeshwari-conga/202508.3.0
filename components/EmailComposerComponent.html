Description

The Email Composer component provides a reusable form for composing and sending emails. It includes email fields (To, CC, BCC), subject, message, template selection, and file attachments.

Inputs

config
Type : EmailFormConfig

Unified configuration object for the email composer. This comprehensive config object contains user info, default emails, file types, field configurations, and initial data. See EmailFormConfig for the structure of this configuration object.
record
Type : any

Record containing primary contact information for pre-populating email fields. This object should contain contact details like PrimaryContact.Email and PrimaryContact.Id which will be used to automatically set the default 'To' email address and fetch additional contact information from the API.

Outputs

emailFormEvent
Type : EventEmitter

Unified event emitter for all email composer events including data changes, template actions, and file operations. This single output handles all component interactions like email data updates, template selections, file uploads/removals, and attachment actions, providing a centralized way to respond to all email composer activities. See EmailFormEvent for event structure
