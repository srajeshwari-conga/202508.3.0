Description

The Send for Signature component is used to send out documents for signature to recipients.

This component provides a comprehensive 3-step workflow for document signature requests:

  1. Document Selection: Choose documents to be sent for signature
  2. Recipients Management: Add and configure signature recipients .
  3. Email Composition: Customize email subject, message, and send the signature request

Supported Signature Providers

-CongaSign: Conga's native digital signature solution with background mode support, defaults to CongaSign if not specified.

-WetSign: Traditional wet signature workflow for physical document signing
