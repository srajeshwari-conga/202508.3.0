Description

This component allows users to filter favorites, for the active pricelist, based on the visibility scope.

Component Behavior:

  • Initially shows only a "Browse Favorite" quick link when showScopeOptions is false
  • After clicking the "Browse Favorite" link, displays scope selection radio buttons
  • Only visible when user is logged in and favorites feature is enabled

Scope Options: Possible values of scope are:

  • 'private': Returns only private favorites that are created by the current user.
  • 'public': Returns all favorites marked as 'public'.
  • 'all': Includes both private and public favorites (default selection).

Inputs

showScopeOptions
Type : boolean
Default value : false

Boolean value to show/hide the scope options for filtering favorites. Scope determines the visibility of the favorites. Scope options include 'private', 'public', and 'all'. Default value is false.

Outputs

onFavoriteScope
Type : EventEmitter<Array<FieldFilter>>

Event emitter to emit the filters to be applied for favorites. This event is triggered whenever the scope of the favorites filter changes. It emits a list of favorite filters of FieldFilter instance. See FieldFilter for more details on the structure and fields required.
