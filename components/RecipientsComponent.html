Description

RecipientsComponent - A comprehensive component for managing document signature recipients

This component provides a full-featured interface for managing recipients in document signature workflows. It supports adding, editing, removing, and reordering recipients with drag-and-drop functionality. The component includes real-time validation, user search capabilities, and flexible signing order management.

Key Features

  • Recipient Management: Add, edit, and remove recipients with comprehensive validation
  • User Search: Search for users via API integration with debounced search functionality
  • Drag & Drop: Reorder recipients using Angular CDK drag-and-drop
  • Signing Order: Optional sequential signing order management
  • Real-time Validation: Instant field validation with user-friendly error messages
  • Internationalization: Full i18n support with ngx-translate
  • Accessibility: WCAG compliant with proper ARIA attributes
