Description

Service for managing eSignature operations including user search, configuration retrieval, and document sending

This service provides methods to interact with the eSignature API for various signature-related operations. It supports multiple signature providers like WetSign and CongaSign, and handles user search functionality with proper error handling and data transformation.

```typescript
// Search for users
this.sendForSignatureService.searchUsers('john').subscribe(users => {
  console.log('Found users:', users);
});

// Get configuration this.sendForSignatureService.getConfiguration().subscribe(providers => { console.log('Available providers:', providers); });

// Send for signature const payload: SendForSignaturePayload = { TemplateId: '', Status: 'Draft', ObjectId: 'quote-123', ObjectType: 'Quote', // ... other properties }; this.sendForSignatureService.sendForSignature('CongaSign', payload).subscribe(result => { console.log('Signature request sent:', result); });


See {@link ESignUser} for user search result structure
See {@link SignProvider} for provider configuration structure
See {@link SendForSignaturePayload} for signature request payload structure
See {@link SignatureRecipient} for recipient structure
See {@link SignatureEmailContent} for email configuration structure
See {@link UserRole} for user role structure
